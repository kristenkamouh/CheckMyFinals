Help & Support

Everything you need to know about using the Exam Scheduler

Getting Started

Follow these simple steps to search for your exam schedule:

  1. Upload Your Schedule File: Click the upload area and select your exam schedule file (.csv or .xlsx format)
  2. Wait for Processing: The file will be parsed automatically in your browser
  3. Search: Enter a course code (e.g., "CSC 212") in the search box
  4. View Results: Your exam details will appear in the results table

Supported File Formats

The application accepts the following file formats:

  • .xlsx - Excel 2007 and later
  • .xls - Excel 97-2003
  • .csv - Comma-separated values

Frequently Asked Questions

Q: Is my exam schedule data safe?
Yes! All file processing happens locally in your browser. We do not upload, store, or transmit your data to any servers. Your information stays on your device.
Q: What if my file won't upload?
Make sure your file is in a supported format (.csv, .xlsx, or .xls). If the problem persists, try converting your file to CSV format using a spreadsheet application.
Q: Why aren't my search results showing?
Double-check that you've entered the course code correctly. The search is case-insensitive, but make sure the course code exists in your uploaded file. Try searching with just the department prefix (e.g., "CSC" instead of "CSC 212").
Q: Can I search multiple courses at once?
Currently, you can search for one course at a time. However, using a department prefix (like "MTH") will show all matching courses from that department.
Q: Do I need to create an account?
No account is needed! Simply upload your schedule file and start searching immediately.
Q: Will my uploaded file be saved?
No. The data is stored temporarily in your browser's memory and is cleared when you refresh or close the page. You'll need to re-upload your file each time you use the application.
Q: Can I use this on my phone or tablet?
Yes! The application is fully responsive and works on mobile devices, tablets, and desktop computers.
Q: What if the exam information looks wrong?
Always verify exam schedules with official university sources. This tool displays the information from your uploaded file as-is. If you notice discrepancies, check that you've uploaded the correct, most recent schedule file.

Troubleshooting

If you encounter issues:

  • Try refreshing the page and uploading your file again
  • Clear your browser cache and cookies
  • Try a different browser (Chrome, Firefox, Safari, Edge are all supported)
  • Ensure your file format is supported (.csv, .xlsx, .xls)
  • Check that your file contains properly formatted exam schedule data

Browser Compatibility

This application works best on modern browsers:

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Safari (latest version)
  • Microsoft Edge (latest version)

Still Need Help?

If you're still experiencing issues or have questions not covered here, please ensure you're using the latest version of your web browser and that JavaScript is enabled.
